Régression Linéaire

y = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ + ε

La régression linéaire modélise la relation entre une variable dépendante y et une ou plusieurs variables indépendantes x.

Hypothèses :
  • Linéarité
  • Indépendance des erreurs
  • Homoscédasticité
  • Normalité des résidus

Random Forest

Random Forest est un algorithme d'ensemble qui combine plusieurs arbres de décision pour améliorer la précision et réduire le surapprentissage.

Avantages :
  • Robuste au surapprentissage
  • Gère bien les valeurs manquantes
  • Fournit l'importance des variables
Paramètres Clés :
n_estimators: Nombre d'arbres
max_depth: Profondeur maximale des arbres
min_samples_split: Nombre minimum d'échantillons pour diviser un nœud

K-Means

J = Σᵢ Σⱼ ||xᵢ - μⱼ||²

K-means est un algorithme de clustering qui partitionne n observations en k clusters, où chaque observation appartient au cluster avec la moyenne la plus proche.

Étapes :
  1. Initialisation aléatoire des centroids
  2. Attribution des points au centroid le plus proche
  3. Mise à jour des positions des centroids
  4. Répétition jusqu'à convergence

Questions Fréquentes

Vous pouvez importer vos données en format CSV, Excel ou JSON via la page d'importation. Glissez-déposez simplement votre fichier ou utilisez le bouton de sélection de fichier.

Le choix de l'algorithme dépend de votre objectif :
  • Pour prédire une valeur continue : utilisez la régression
  • Pour prédire une catégorie : utilisez la classification
  • Pour grouper des données similaires : utilisez le clustering

Les résultats sont présentés sous forme de métriques et de visualisations :
  • Métriques de performance (accuracy, precision, recall, etc.)
  • Graphiques de performance
  • Matrices de confusion pour la classification
  • Courbes d'apprentissage
