Régression Linéaire

y = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ + ε

La régression linéaire modélise la relation entre une variable dépendante y et une ou plusieurs variables indépendantes x.

Applications :
  • Prévision des ventes
  • Analyse des tendances économiques
  • Modélisation des coûts
Exemple :
                            Input: x = [1, 2, 3], y = [2.1, 4.1, 6.1]
                            Output: y = 2.0x + 0.1

Régression Logistique

La régression logistique est utilisée pour modéliser une variable dépendante binaire (0 ou 1) en fonction des variables indépendantes.

Applications :
  • Analyse des probabilités de maladie
  • Détection de spam
  • Prédiction de churn client

Random Forest

Random Forest est un algorithme d'ensemble basé sur plusieurs arbres de décision pour une classification ou une régression précise.

Cas d'usage :
  • Détection de fraudes
  • Analyse des données médicales
  • Systèmes de recommandation
Exemple :
                            Input: Features = [Age, Income], Label = [Yes/No]
                            Output: Probabilité de la classe = 0.85 (Yes)

Support Vector Machine (SVM)

SVM est un algorithme puissant pour la classification qui maximise le "margin" entre les classes.

Applications :
  • Reconnaissance faciale
  • Analyse sentimentale
  • Prédiction de maladies

K-Means

K-means regroupe les données en clusters homogènes basés sur leurs similarités.

Avantages :
  • Simplicité et rapidité
  • Convient à de grandes quantités de données
Exemple :
                            Input: Points = [(1,2), (2,3), (8,9)]
                            Output: Clusters = {[Cluster 1: (1,2), (2,3)], [Cluster 2: (8,9)]}

DBSCAN

DBSCAN est un algorithme de clustering basé sur la densité, capable de détecter des clusters de formes arbitraires.

Applications :
  • Analyse des anomalies
  • Groupement des comportements utilisateurs

Questions Fréquentes

Vous pouvez importer vos données en format CSV, Excel ou JSON via la page d'importation. Glissez-déposez simplement votre fichier ou utilisez le bouton de sélection de fichier.

Le choix de l'algorithme dépend de votre objectif :
  • Pour prédire une valeur continue : utilisez la régression
  • Pour prédire une catégorie : utilisez la classification
  • Pour grouper des données similaires : utilisez le clustering

Les résultats sont présentés sous forme de métriques et de visualisations :
  • Métriques de performance (accuracy, precision, recall, etc.)
  • Graphiques de performance
  • Matrices de confusion pour la classification
  • Courbes d'apprentissage
